# Why These Files Are Archived

The files in this folder (`01-setup.md` through `06-claudecode-guide.md`) were
build guides used to scaffold ListenRoom from an empty directory.

The app is now fully built and running. Asking Claude Code to re-read all six
of these on every session is wasteful — they describe *how to build* the app,
not *how the app works*.

## What to read instead

| What you need | Where to find it |
|---|---|
| How the app works right now | `guidance/1.0.0.md` |
| Full user flows and data paths | `docs/flow.md` |
| How to install, run, and debug | `README.md` |
| Hard rules and project conventions | `CLAUDE.md` |

## When these files might still be useful

- Rebuilding the monorepo workspace from scratch
- Understanding why specific architectural choices were made
- Reference for the original design intent (before the app evolved)
